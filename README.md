# 🐉 Delver's Grimoire: DM Co-Pilot (Masterwork Edition)

An authentic, AI-powered collaboration tool designed to automate the administrative heavy lifting of D&D 5e dungeon mastering.

## 🚀 Live App
Check out the fully deployed version here: **[www.delversgrimoire.com](https://www.delversgrimoire.com/)**

## ✨ Core Features
- **📊 Visual Encounter Architect:** Plots monsters on a CR/HP scatter chart to help balance combat encounters visually.
- **🏰 Procedural Dungeon Maps:** Generates tactical ASCII grids with AI-driven environmental descriptions.
- **🎭 NPC Quick-Forge:** Instantly creates NPCs with physical descriptions and secret motivations.
- **📥 Adventure Export:** **(New!)** Download your entire generated session as a clean, organized Markdown file.
- **🎲 Integrated Dice Utility:** Sidebar dice rollers ($d20$, $d12$, $d10$, $d8$, $d6$, $d4$) for quick math.
- **💰 Loot & Hoard Generators:** CR-scaled treasure calculations with flavor text.

## 🛠️ Technical Stack
- **Frontend:** Streamlit
- **AI Engines:** Groq (Cloud) & Ollama (Local)
- **Database:** Google Firestore (Persistent Analytics)
- **Data Export:** Markdown / Pandas
